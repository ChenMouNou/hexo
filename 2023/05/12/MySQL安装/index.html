环境变量

右击主机点击属性，win7直接点击环境变量。win10会进入系统关于，在右边的相关设置点击高级设置再点击环境变量。把你的mysql路径下的bin目录添加进去。

然后别忘了按确定。

测试

按win+r输入cmd

输入mysql -u root -p 再输入你之前设置的密码，就可以了

连接

打开Navicat for MySQL（其他的也行）

点击连接 >MySQL>

输入密码，端口号没改过的不用改，连接名随便写，localhost是本机的意思不用改。

测试下连接，成功就OK了。

启动和关闭

使用管理员打开cmd（win+q搜索cmd右击以管理员运行）输入net start mysql80启动（mysql服务具体叫什么看你的任务管理器最后面的服务，随便一个服务点击它的名称切英文输入法，输入mysql就可看到了）

管理员cmd输入net stop mysql80关闭服务
